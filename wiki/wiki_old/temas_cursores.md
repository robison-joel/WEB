Temas para cursores
================================

Breeze
---------------------------------------

![Cursors](./pics/breeze_cursors.png)

Atualize os repositórios

`sudo apt update`

Instale o tema de cursores

`sudo apt install -y breeze-cursor-theme`

Oxygen
---------------------------------------

![Cursors](./pics/oxygen_sursors.png)

Atualize os repositórios

`sudo apt update`

Instale o tema de cursores

`sudo apt install -y oxygen-cursor-theme`
